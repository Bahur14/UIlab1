Лабораторна робота №1: Програмування Web-інтерфейсів

Виконав(ла): Бахурінський Олександр Дмитрович
Група: КВ-52мп

Завдання
Розробити клієнтську частину Web-додатку з використанням фреймворка Javascript/ReactJS та CSS-фреймворку (Tailwind CSS). 

Тематика додатку: Список справ (To-Do List). 
Реалізовано функціонал створення, редагування, видалення та перегляду завдань, а також сторінки реєстрації, входу, профілю та інформації про додаток. Дані зберігаються у localStorage браузера.
